## 🎯 Purpose
> The goal of this project is to present my work in a professional way and demonstrate my ability to build modern, scalable, and user-friendly web applications.

## ✨ Features
* 🌐 **Multi-language support:** Arabic & English
* 🌙 **Theming:** Light and Dark mode support
* 🧑‍💻 **Projects Showcase:** A dedicated section for my projects
* 🛠️ **Tech Overview:** Skills and technologies overview
* 📱 **Responsive:** Fully responsive design for all screen sizes
* ⚡ **Performance:** Fast and optimized performance
* 🎨 **Modern UI:** Clean and modern user interface
* 📬 **Contact:** Easy-to-use contact section
